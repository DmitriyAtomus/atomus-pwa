# UI-аудит CRM «Атомус» (atomus-pwa)

Дата: 2026-06-16. Бэкап: `projects\_backup_atomus-pwa_20260616_151307`.

## Архитектура
- **Главное приложение:** `index.html` (~600 КБ, SPA) + общий `app.css` (627 КБ) + `app-*.js`.
- **Отдельные страницы** (свои стили, НЕ подключают `app.css`): `syrovarnya/index.html`, `atomcad/*`, `kp/*`.
- В `app.css` есть система токенов в `:root` (брендовый цвет `--brand:#2D5F8B`, текст, статусные цвета, тени).

## Главная проблема: ДВЕ параллельные системы стилей
- **Токены** (`app.css :root`) построены на бренд-синем **#2D5F8B**.
- **Инлайн-стили в index.html** используют ДРУГУЮ палитру (Tailwind-sky): доминирует **#0c4a6e (82×)**, плюс #0369a1, #bae6fd, #f0f9ff. → два разных синих по сайту.
- Итог: 289 хардкод-hex прямо в разметке `index.html` мимо токенов.

## 1. Цвета
- index.html: **289** инлайн-hex, топ: `#0c4a6e`×82, `#e2e8f0`×20, `#475569`×18, `#f0f9ff`×13.
- `syrovarnya/index.html`: **231 уникальный цвет** в собственном `<style>` (отдельная вселенная).
- Рекомендация: одна палитра через токены; инлайн-цвета → на переменные.

## 2. Скругления (border-radius) — сильный разнобой
- `app.css`: ~20 разных значений (8,10,12,14,6,4,3,16,9,5,7,11,2,1,15,18,20px, 999px, 50%, 0).
- index.html инлайн: ещё 0,4,6,8,10,12,18,999px.
- Рекомендация: шкала из 5 — **4 / 8 / 12 / 16 px + 999px(пилюля) + 50%(круг)**.

## 3. Шрифты (font-size) — нет типографической шкалы
- index.html: **215** инлайн font-size, дробные значения: 10.5 / 11 / 11.5 / 12 / 12.5 / 13 / 13.5 / 14px…
- Рекомендация: шкала **12 / 13 / 14 / 16 / 18 / 22 / 24 px** (убрать .5-значения).

## 4. Отступы (padding) — не по сетке
- Инлайн: 3 / 5 / 6 / 7 / 8 / 10 / 12 / 14 / 16 px вперемешку.
- Рекомендация: сетка кратно 4 — **4 / 8 / 12 / 16 / 24 px**.

## 5. Кнопки — нет единой системы
- **40+ классов** кнопок (.btn-primary, .btn-secondary, .back-btn, .cal-delete-btn, .calc-mode-btn, .emp-pwd-btn…).
- Использование в index.html: `.btn-secondary`×72, `.btn-primary`×45, `.btn-small`×29, `.back-btn`×21.
- **43 кнопки** со собственным инлайн-стилем.
- Рекомендация: одна база `.btn` + модификаторы `-primary/-secondary/-danger/-ghost` и размеры `-sm/-lg`; одиночные классы свести к ним.

## 6. Отдельные страницы
- `syrovarnya` (231 цвет!), `atomcad/editor`, `atomcad/wizard`, `kp/kamera-7x10` — каждая со своими стилями.
- Рекомендация: после наведения порядка в главном app — подключить общие токены/кнопки и сюда.
